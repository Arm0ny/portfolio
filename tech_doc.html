Conditions

Now that we know well how to use operators to perform operations we can talk about Conditional Statements, these are special functions provided by every programming language that are used to perform an application Only if a certain Condition is met. The principal "building blocks" of a condition in python are:

  • if
  • else

Let's see them in action with an example:
let's assume that you want to check if a certain number a is grater than another number b and if this is true, you whant to display a message which says that a is grater than b, if not you want to display the opposit in this case we can do as it follows:

a = 26
b = 10
if a > b:

print("a is grater than b")

else:

print("a is not grater than b")

 Output: a is grater than b

The if statement takes care only of the condition expressed after it's called while the else statement does something in all the other cases, infact, in our example, the if statement checks only if a in grater than b and the else statement is "activated" in all the other cases (a < b or a = b).

Loops

As the name suggests Loops are special constructs provided by Python for repeating a certain action in a special condition. The main two loops of the python programming language are:

  • While Loop
  • For Loop

the While Loop is used to repeat an instruction unless a certain condition is met, the condition itself is expressed after the While keyword, the instruction or instructions to repeat are expressed after the semicoloumn whith the correct indentation. Let's make an example:

a = 0
b = 5
while a < b:

print(a)

a += 1

The code itself is almost self explainatory, we declaired two variables: one is our a variable and the other is our b variable, entering the loop, the program checks is a is less than b, and if the Condition is met, it prints the value of a and increments it (a *= 1 is the shorthand for a = a + 1) these two instructions are repeted 5 times aka unless the value of a is less than the value of b

Output_1 0
Output_2 1
Output_3 2
Output_4 3
Output_5 4

Just like yhe While loop, the For Loop is also used to repeating actions subdued to a condition whith the difference that it can be used to repeat in a certain range of times and the word is not higlighted casually, infact, the for construct is often used whith the range() function. let's write some code to be clear:

a = 0
for a in range(0,5,1)

print(a)

Ok now that wer have our piece of code to analyze, we can see in first place that we haven't used the instruction a += 1, this is because the for loop automatically increments the variable provided in it's call. But how he knows the way to increment the variable and where to stop and begin? All those data are provided by the range() function, the first argumrnt in the parenthesis is the beginning of the interval, the second is the end and the third is the increment. So for a verbose lectur of the code we can say that in the loop we print the value of a 0 to 5 times and each time we increment a by 1. Note that the beginning and ending of the interval are independent from the value of a and indicates only the number of time to repeat the cicle
